Admin Credentials

Admin email and password are managed via environment variables in backend/.env. Update ADMIN_EMAIL and ADMIN_PASSWORD to change credentials. Restart the Flask server after changes.

Session Expiry

Admin JWT tokens expire after ADMIN_JWT_EXPIRY_HOURS hours (default: 8). After expiry you will be automatically redirected to the login page.

Database

Connected to MongoDB database: data_integrity_platform. Collections in use: users, data_records, otp_verifications, audit_logs, rate_limits.

Security Notes

All /api/admin/* routes require a valid admin JWT. The token is checked for role: "admin" on every request. Non-admin tokens receive 403 Forbidden.
